# Theme Toggle App (React)

A minimal React app demonstrating a theme toggle (Light/Dark) using state-driven UI.

Features
- Theme managed with `useState()`
- Global background & text change via `useEffect()`
- Current theme displayed
- Theme passed as props to a child component
- Conditional rendering based on theme

Quick start

1. Install dependencies:

   npm install

2. Run development server:

   npm run dev

Open the dev server URL shown in the terminal (Vite default: http://localhost:5173).

Notes
- This project is intended as a compact example for learning React state, props, and conditional rendering.
